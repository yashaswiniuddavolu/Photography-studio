# LensCraft Studio - Full-Stack Photography Platform

![Full-Stack Photography Platform](https://via.placeholder.com/800x400/2d3748/FFFFFF?text=LensCraft+Studio+Full-Stack)

A complete full-stack photography management platform with React frontend, Node.js/Express backend, and MongoDB database. Features client management, gallery system, booking, and admin controls.

## üöÄ Tech Stack

### **Frontend**
- **React 18** with Vite for fast builds
- **Tailwind CSS** for styling
- **React Router** for navigation
- **React Hooks** for state management
- **Axios** for API calls

### **Backend**
- **Node.js** with Express.js
- **MongoDB** with Mongoose ODM
- **JWT Authentication** for secure login
- **Multer** for image uploads
- **Nodemailer** for email notifications

### **Dev Tools**
- **Git** for version control
- **Postman/Insomnia** for API testing
- **MongoDB Atlas/Compass** for database
